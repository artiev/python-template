Python Project Template
=======================

All you need to kick-start a python project should be here, as the template
includes code snippets, configuration files and makefile to spend more time
on your application and less on the setting up the environment.

This small project aims at offering a personal take on formalizing python
project structures, together with good development practices and tools. The
main point here is setting up the most important tools a developer needs,
and ensure they function out of the box so starting any new project can be
done in a breeze.
